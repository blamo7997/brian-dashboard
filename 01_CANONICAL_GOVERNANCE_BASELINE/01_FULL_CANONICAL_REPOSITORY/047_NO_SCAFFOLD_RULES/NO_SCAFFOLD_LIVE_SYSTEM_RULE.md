# NO SCAFFOLD / LIVE SYSTEM RULE™

Date: 20260610-220357
Status: Canonical

## Rule
Do not intentionally create scaffolds, placeholders, throwaway systems, fake implementations, or known-short-lived structures when the objective is live, production-bound, durable, or long-term.

## Production Direction
When going live, build toward the intended durable implementation.

## Temporary Measures
Temporary measures are allowed only for:
- Emergency containment
- Safety
- Security
- Recovery
- Testing in clearly isolated non-production context

## Temporary Measures Must Include
- Owner
- Reason
- Risk
- Expiration or replacement plan
- Native/durable conversion path
- Audit note
- Archive note

## Hard Rule
No fake final systems.
No permanent temporary systems.
No scaffold treated as live architecture.
