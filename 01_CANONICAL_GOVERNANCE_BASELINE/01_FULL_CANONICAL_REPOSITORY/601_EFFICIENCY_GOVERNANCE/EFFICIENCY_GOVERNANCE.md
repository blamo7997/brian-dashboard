# EFFICIENCY GOVERNANCE™

Date: 20260610-223134
Status: Canonical Direction

## Goal
Preserve everything without forcing every future project to load everything.

## Laws
- Pointer before copy.
- Index before search.
- Summary before full file.
- Active before archive.
- Canonical before historical.
- Rehydrate only when necessary.
- Export compact by default.
- Export full only when required.

## Preservation Constraint
Optimization must not delete, erase, or silently replace protected continuity.
