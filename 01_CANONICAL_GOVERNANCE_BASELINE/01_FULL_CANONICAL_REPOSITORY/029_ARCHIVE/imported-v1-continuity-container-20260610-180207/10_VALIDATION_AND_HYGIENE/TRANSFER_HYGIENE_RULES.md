# TRANSFER HYGIENE RULES™

Status: Mandatory
Inheritance: Mandatory
Date: 20260610-173014

## Canonical Transfer Files Must Contain
- Project name
- Current state
- Stopped-here marker
- Approved decisions
- New rules
- Open questions
- Gaps and risks
- Next work
- Continuity rules

## Canonical Transfer Files Must Not Contain
- PS prompts
- Console noise
- Accidental pasted shell markers
- Duplicate blocks
- Old logs unless needed as evidence
- Repeated architecture lists
- Unrequested architecture expansion

## Hard Rule
Clean continuity beats giant messy handoffs.
