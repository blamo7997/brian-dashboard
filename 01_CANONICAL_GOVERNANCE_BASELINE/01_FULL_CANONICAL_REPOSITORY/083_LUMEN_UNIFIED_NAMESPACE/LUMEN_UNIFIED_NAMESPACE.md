# LUMEN UNIFIED NAMESPACE™

Date: 20260610-222040
Status: Canonical Direction

## Purpose
Prevent disconnected naming across OS, website, Vault, and bridge systems.

## Namespace Pattern
lumen://domain/type/id

## Examples
- lumen://identity/user/USER-ID
- lumen://vault/object/VAULT-ID
- lumen://website/session/SESSION-ID
- lumen://os/workspace/WORKSPACE-ID
- lumen://bridge/transfer/TRANSFER-ID
- lumen://policy/rule/RULE-ID

## Rule
Names must support continuity, lineage, and cross-channel lookup.
