## Duplicate Control Rules
- One canonical authority per subject.
- Historical duplicates become references.
- Archive duplicates remain preserved but not active.
- New data becomes delta.
- Search index references records instead of copying them.
- Export packet includes pointers instead of full histories.

## Rule
Preserve without multiplying active truth.
