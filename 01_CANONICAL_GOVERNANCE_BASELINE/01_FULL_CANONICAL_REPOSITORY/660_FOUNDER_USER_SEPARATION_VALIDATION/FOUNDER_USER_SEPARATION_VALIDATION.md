# FOUNDER / USER SEPARATION VALIDATION™

Date: 20260610-224911
Status: Required Validation

## Must Always Validate
- Founder-only data is not visible to normal users.
- User data is sandboxed from other users.
- Website-only users cannot access OS founder controls.
- OS users cannot bypass website/Vault permissions.
- Lumen website edits require founder/admin authority.
- Vault records are permission-scoped.
- Web results do not leak private context.
- Timestamping does not expose private information.
- Export packets do not expose founder-only material to unauthorized users.

## Rule
Connected ecosystem does not mean shared access.
