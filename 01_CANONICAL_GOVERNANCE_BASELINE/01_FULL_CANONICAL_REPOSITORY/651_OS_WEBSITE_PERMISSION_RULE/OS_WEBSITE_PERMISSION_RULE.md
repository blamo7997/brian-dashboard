# OS / WEBSITE PERMISSION RULE™

Date: 20260610-224911
Status: Hard Rule / Cross-Project

## Rule
Lumen OS connects to the website.

The website may also provide a website-version Lumen experience for users who do not use the OS.

Both paths must use appropriate user permissions.

## Separation Rule
Users must never gain access to Brian/founder-only:
- OS actions
- Website administration
- Vault records
- Founder dashboards
- Governance tools
- Private activity
- Private files
- Private interactions
- Private continuity records
- Business-critical internal controls

## Access Channel Rule
OS, website, Vault, ChatGPT/OpenAI bridge, and future Lumen-native AI are connected channels, but connection does not mean shared permissions.

Connected ecosystem.
Separated authority.
Permissioned access.
Vault-first memory.
