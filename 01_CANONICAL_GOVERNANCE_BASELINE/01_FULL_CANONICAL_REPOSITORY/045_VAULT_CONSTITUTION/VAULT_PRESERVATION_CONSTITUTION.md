## Purpose
Define the Vault as the protected continuity and evidence memory layer.

## Vault Preserves
- Decisions
- Rules
- Standards
- Directives
- Evidence
- Lineage
- History
- Transfers
- Manifests
- Validation records
- Approvals

## Vault Rule
The Vault is not a casual file bin.
The Vault is protected continuity memory.

2. Reference
3. Archive
4. Modify with evidence
5. Delete only by justified exception
