# VETTED WEB RESULT POLICY™

Date: 20260610-224911
Status: Hard Rule / Information Quality

## Rule
When users request external or current information, Lumen should scan the web and return vetted, verified, permission-safe results.

## Result Requirements
- Relevant
- Current when required
- Source-aware
- Permission-safe
- Not exposing private Vault records
- Not exposing founder-only data
- Suitable for the user's role and entitlement
- Stored/referenced in Vault when appropriate

## Preferred Source Order
1. Official sources
2. Primary sources
3. Reputable expert sources
4. Cross-verified secondary sources
5. Lower-confidence sources only if clearly labeled or excluded

## Rule
Return verified results, not raw internet noise.
