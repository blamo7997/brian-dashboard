## Confirmed Required Rules Added
- OS connects to website.
- Website version of Lumen is supported.
- Website-only users are permission-bounded.
- Users never access Brian/founder-only OS, website, Vault, governance, or private activity.
- Lumen can edit website/ecosystem records only through proper authority.
- Every edit/action should be timestamped.

- Web is scanned when current/external information is needed.
- Results must be vetted, verified, and permission-safe.
