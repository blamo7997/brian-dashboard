# LUMEN WEBSITE OBJECT MODEL™

Date: 20260610-222040
Status: Canonical Direction

## Purpose
Define the website as a full Lumen access channel, not a secondary or disconnected product.

## Website Object Types
- Website User
- Website Session
- Website Dashboard
- Website Portal
- Website Purchase
- Website Entitlement
- Website Upload
- Website File
- Website Interaction
- Website Notification
- Website Chat
- Website Vault Link
- Website OS Link
- Website Bridge Link

## Website-Only Rule
Users may use Lumen only through the website.
This path must remain supported.

## Required Fields
- WebsiteObjectID
- IdentityID
- VaultLink
- EntitlementLink
- ConsentLink
- SecurityProfile
- AccessibilityProfile
- LocalizationProfile
- Status
- Version
- Lineage
- RecoveryPath

## Rule
Website records must connect into the same continuity-governed ecosystem as OS and Vault records.
