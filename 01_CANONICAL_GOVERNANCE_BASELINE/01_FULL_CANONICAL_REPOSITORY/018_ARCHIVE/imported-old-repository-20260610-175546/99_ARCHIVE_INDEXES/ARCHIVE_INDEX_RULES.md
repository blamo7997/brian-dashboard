# ARCHIVE INDEX RULES™

Status: Mandatory
Inheritance: Mandatory
Date: 20260610-173014

## Purpose
Old project-transfer material should not be deleted.
It should be indexed, summarized, and archived.

## Archive Rule
If something is superseded, keep the record and mark it superseded.
Do not silently remove it.

## No Duplicate Rule
If the same content exists, reference the original instead of copying it again.
