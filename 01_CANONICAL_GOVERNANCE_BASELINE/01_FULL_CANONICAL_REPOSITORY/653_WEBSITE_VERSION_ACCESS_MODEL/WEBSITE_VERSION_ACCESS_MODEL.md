# WEBSITE VERSION ACCESS MODEL™

Date: 20260610-224911
Status: Canonical Direction

## Purpose
Support users who want to use Lumen only through the website.

## Website-Version Users Receive
- Permissioned dashboard
- Website-safe Lumen tools
- Vault-backed continuity where consented
- Accessibility
- Localization
- Entitlements
- Recovery
- Vetted results
- User-specific saved context where allowed

## Website-Version Users Do Not Receive
- Founder OS access
- Founder website admin access
- Internal Vault access
- Other user access
- Governance override authority
- Hidden system controls

## Rule
Website-only Lumen is first-class, but permission-bounded.
