# LUMEN INSIDE-ONLY CONTROL STANDARD

Founder and users should not have to leave Lumen for normal operations.

PowerShell, terminal commands, scripts, and files are bootstrap tools only.

Target behavior:
- Open Lumen.
- Type or speak what you want.
- Lumen interprets it.
- Lumen checks memory.
- Lumen checks existing code.
- Lumen checks duplicates.
- Lumen applies Guardian rules.
- Lumen runs audits/tests/builds.
- Lumen provides a brief result.
- User can view step-by-step only if wanted.

Inside Lumen actions:
- Run audit
- Run tests
- Run build
- Install safe expansions
- Review blocked expansions
- Approve/reject changes
- Create portal
- Create dashboard
- Create product/service/membership
- Add job/industry/position
- Change website/theme
- Generate website
- View Registry
- View Vault
- View Knowledge Graph
- View Digital Twin
- View user memory
- View rollback/recovery status

Rule:
Normal operation should happen from the Lumen interface, not outside Lumen.
