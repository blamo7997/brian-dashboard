# WINDOWS 11 EXISTING OS MODE

Lumen install mode:
- Existing OS mode
- Runs over Windows 11
- Uses a workspace folder
- Uses desktop launcher
- Uses rollback/recovery
- Uses update-on-launch
- Does not replace Windows
- Does not overwrite boot files
- Does not modify partitions

Future possible options:
- Existing Windows app
- Portable workspace app
- Signed installer
- System tray runtime
- Local service
- Full separate OS installer only with expert review, recovery media, and explicit consent
