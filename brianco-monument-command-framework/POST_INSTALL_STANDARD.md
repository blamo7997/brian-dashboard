# Brian & Co Post-Install / Update Standard

After any approved software, app, service, tool, resource, OS component, website component, chatbot feature, portal feature, integration, or device layer is installed or updated:

1. Verify install integrity.
2. Confirm free/trusted source.
3. Check compatibility.
4. Run build/test where applicable.
5. Run accessibility/localization review.
6. Run security/proprietary review.
7. Run integration readiness check.
8. Debug detected errors.
9. Apply safe fixes where authorized.
10. Retest.
11. Repeat until zero detected errors or documented blocker.
